### Reflections
[Weekly reflections](../../resources/reflections.md) are due by 8am Saturday. Please submit your reflection via commit to a branch on this repository and submit a pull request when you are ready to share it. [How to submit your work](../../resources/submitting_your_work_via_github.md)

This week, in addition to your *** Weekly Whats:tm:*** ; write an excellent commit message for your reflections. Follow the guidelines for [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) and summarize the content of your reflection in commit message. For example:

```
ADD: reflection for week 3

This week's reflection focussed primarily on the lessons learned about git and GitHub. I learned about the importance of utilizing the commit message to tell the story of the code, to both my teammates, and to future me. 

In addition to the git work, we started digging deeper into learning Ruby on Rails. We are building two apps along with the tutorial series. Plus we aer each building our own 'breakable toy'. I am building a web app that will help me keep track of my 'magic' that is Rails. I am excited to get started on it!!




```
